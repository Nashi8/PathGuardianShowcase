Overview

Path Guardian is an AI-driven web application designed to improve urban safety for cyclists and stationary drivers by enabling real-time reporting and visualization of road hazards. Using generative AI, the app delivers intelligent alerts and prompts to guide user interactions while promoting safer commuting environments.

My Role & Key Contributions

  • Prompt Engineering & UX Design: Created AI-powered conversational flows using OpenAI API for onboarding and safety prompts.
  • Interface Development: Built the app UI using Streamlit, including a warning acknowledgment system.
  • Usability Testing: Conducted testing to refine workflows, layout, and clarity of tutorial guidance.
  • Demo & Documentation: Produced tutorials and interface documentation for user support and stakeholder presentations.

Technologies Used

  • OpenAI API (generative text + prompt workflows)
  • Streamlit (interactive frontend development)
  • Python (backend scripting and app logic)
  • GitHub Copilot (AI-powered coding assistant)
  • HTML/CSS (page and interface customization)
